# Fund Agent - 数据仓库

这是 [my-fund-agent-v2](https://github.com/leahliu777/my-fund-agent-v2 ) 项目的数据仓库。

## 说明

- 本仓库由 GitHub Actions 自动更新
- 包含每日生成的基金监控报告和数据
- 通过 GitHub Pages 提供访问

## 访问

- 最新报告：https://leahliu777.github.io/my-fund-agent-v2-data/
- 历史报告：https://leahliu777.github.io/my-fund-agent-v2-data/reports/

## 更新频率

每天4次：9:30, 13:30, 16:00, 22:10（北京时间 ）
